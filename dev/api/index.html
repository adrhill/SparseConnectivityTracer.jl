Interface

SparseConnectivityTracer.connectivity_pattern — Function
connectivity_pattern(f, x)
connectivity_pattern(f, x, T)

Enumerates inputs x and primal outputs y = f(x) and returns sparse matrix C of size (m, n) where C[i, j] is true if the compute graph connects the i-th entry in y to the j-th entry in x.

The type of index set S can be specified as an optional argument and defaults to BitSet.

Example

julia> x = rand(3);

julia> f(x) = [x[1]^2, 2 * x[1] * x[2]^2, sign(x[3])];

julia> connectivity_pattern(f, x)
3×3 SparseArrays.SparseMatrixCSC{Bool, UInt64} with 4 stored entries:
 1  ⋅  ⋅
 1  1  ⋅
 ⋅  ⋅  1

SparseConnectivityTracer.hessian_pattern — Function
hessian_pattern(f, x)
hessian_pattern(f, x, T)

Computes the sparsity pattern of the Hessian of a scalar function y = f(x).

The type of index set S can be specified as an optional argument and defaults to BitSet.

Example

julia> x = rand(5);

julia> f(x) = x[1] + x[2]*x[3] + 1/x[4] + 1*x[5];

julia> hessian_pattern(f, x)
5×5 SparseArrays.SparseMatrixCSC{Bool, UInt64} with 3 stored entries:
 ⋅  ⋅  ⋅  ⋅  ⋅
 ⋅  ⋅  1  ⋅  ⋅
 ⋅  1  ⋅  ⋅  ⋅
 ⋅  ⋅  ⋅  1  ⋅
 ⋅  ⋅  ⋅  ⋅  ⋅

julia> g(x) = f(x) + x[2]^x[5];

julia> hessian_pattern(g, x)
5×5 SparseArrays.SparseMatrixCSC{Bool, UInt64} with 7 stored entries:
 ⋅  ⋅  ⋅  ⋅  ⋅
 ⋅  1  1  ⋅  1
 ⋅  1  ⋅  ⋅  ⋅
 ⋅  ⋅  ⋅  1  ⋅
 ⋅  1  ⋅  ⋅  1

Internals

SparseConnectivityTracer works by pushing Number types called tracers through generic functions. Currently, three tracer types are provided:

SparseConnectivityTracer.ConnectivityTracer — Type
ConnectivityTracer{I,S}(indexset) <: Number

Number type keeping track of input indices of previous computations.

The provided index set type S has to satisfy the following conditions:

  • it is an iterable with <:Integer element type
  • it implements union

Subtypes of AbstractSet{<:Integer} are a natural choice, like BitSet or Set{UInt64}.

For a higher-level interface, refer to connectivity_pattern.

source
SparseConnectivityTracer.JacobianTracer — Type
JacobianTracer{I,S}(indexset) <: Number

Number type keeping track of input indices of previous computations with non-zero derivatives.

The provided index set type S has to satisfy the following conditions:

  • it is an iterable with <:Integer element type
  • it implements union

Subtypes of AbstractSet{<:Integer} are a natural choice, like BitSet or Set{UInt64}.

For a higher-level interface, refer to jacobian_pattern.

source
SparseConnectivityTracer.HessianTracer — Type
HessianTracer{I,S,D}(indexset) <: Number

Number type keeping track of input indices of previous computations with non-zero first and second derivatives.

The provided index set type S has to satisfy the following conditions:

  • it is an iterable with <:Integer element type
  • it implements union

Subtypes of AbstractSet{<:Integer} are a natural choice, like BitSet or Set{UInt64}.

For a higher-level interface, refer to hessian_pattern.

source
